# AIGuidebook

Nettside utviklet som del av PRO1000-gruppeprosjekt ved USN.
Guideboken hjelper studenter med ansvarlig og etisk bruk av kunstig intelligens.

## Mappestruktur

```
aiguidebook/
├── index.html               # Forside
├── README.md                # Denne filen
├── css/
│   └── styles.css           # All CSS-styling
├── js/
│   └── main.js              # JavaScript-funksjonalitet
└── pages/
    ├── tillatt-bruk.html    # Tillatt bruk vs. juks
    ├── kildehenvisning.html # Kildehenvisning APA
    ├── sjekkliste.html      # Sjekkliste før innlevering
    ├── hva-er-ki.html       # Hva er KI?
    ├── personvern.html      # Personvern
    ├── eksamen.html         # Eksamensforberedelse
    ├── faq.html             # Ofte stilte spørsmål
    └── ordliste.html        # Ordliste
```

## Teknologi

- HTML5, CSS3, JavaScript (vanilla)
- Ingen eksterne rammeverk
- Fungerer uten JavaScript (US-NFR14)

## Kjøre lokalt

Åpne `index.html` direkte i nettleseren,
eller bruk Live Server i VS Code.
